# 01 Option Basic

## 1. 为什么需要期权？

期货是线性风险工具，期权是非线性风险工具。期权的核心价值在于，它允许投资者在保留有利方向收益的同时，限制不利方向的损失。

## 2. 期货和期权的区别

| 维度 | 期货 | 期权 |
|---|---|---|
| 权利义务 | 双方都有义务 | 买方有权利，卖方有义务 |
| 收益结构 | 线性 | 非线性 |
| 最大亏损 | 理论上较大 | 买方最大亏损为权利金 |
| 是否支付权利金 | 不支付 | 买方支付权利金 |
| 核心风险 | 方向风险 | 方向、波动率、时间价值等 |

## 3. Call 和 Put 的基本定义

Call：看涨期权，买方拥有未来以约定价格买入标的的权利。

Put：看跌期权，买方拥有未来以约定价格卖出标的的权利。

## 4. 我的理解

期权不是简单的“看涨看跌工具”，而是一种风险结构设计工具。通过期权，可以更精细地管理价格风险、波动率风险和时间价值风险。
