# Week 1 Option Basics - Question List

## Part A：期权为什么存在？

### Q1. 期货已经可以做多做空、套期保值，为什么市场还需要期权？

我的回答：
1. 期货、股票等资产都是线性资产，风险和收益都是无限的，风险收益都是有敞口的，而期权作为非线性资产，期权买方可以承担有限风险而获取无限的利益
2. 期权分为权利方和义务方，权利方在期权到期时有权利选择行使或者弃权，相当于购买保险，最多只损失权利金

改进：

期货已经可以做多、做空和套期保值，但期货是线性风险工具。使用期货套保时，企业可以锁定价格风险，但同时也会锁死有利方向的收益。

期权存在的意义在于，它提供了一种非线性风险结构。期权买方支付权利金后，获得未来在有利情况下行权的权利，在不利情况下可以放弃行权，因此最大亏损通常被限制在权利金以内。

所以，期权不是期货的简单替代品，而是更灵活的风险管理工具。它既可以用于方向性交易，也可以用于保护库存、锁定采购成本、交易波动率和管理尾部风险。

我能联系到的交易/产业场景：
1. 持有标的资产如焦煤期货看涨的时候，为预防标的资产下跌风险，购买jm看跌期权对冲下跌风险，最多只会损失权利金
2. 在持有标的头寸后，遇到黑天鹅而情况下，买入期权作为风险对冲

### Q2. 期权和期货最本质的区别是什么？不要只写“一个有权利一个有义务”，要解释这个区别带来的风险结构变化。

我的回答：

1. 期权的权利与义务性质，作为期权的买方，最大的风险是损失权利金，而受益无限，而期货作为线性资产，风险收益均无限
2. 而作为期权的卖方，最大收益便是期权买方所支付的权利金，而单腿情况下风险无线，因此买方来说，收益有限且固定，风险无限

改进版：

期权和期货最本质的区别是权利义务结构不同，由此导致风险结构不同。

期货交易中，多空双方都有履约义务，因此盈亏随标的价格变化呈线性变化。做多期货时，价格上涨赚钱、下跌亏钱；做空期货时，价格下跌赚钱、上涨亏钱。

期权交易中，买方支付权利金后获得权利而非义务，因此在不利情况下可以放弃行权，最大亏损通常是权利金；卖方收取权利金后承担履约义务，因此最大收益通常是权利金，但在极端行情下可能承担很大亏损。

所以，期权和期货的区别不是简单的交易品种不同，而是风险结构从线性对称变成了非线性不对称。

### Q3. 为什么说期权是“非线性风险工具”？这个非线性具体体现在哪里？

我的回答：期货和股票，收益随着标的价格的变动呈线性变化，而对期权来说，买方最大亏损不会超过权利金，卖方最大收益不会超过权利金，无论价格再变动，二者都不会再发生变化

改进版：

期权是非线性风险工具，因为期权的盈亏并不是随着标的价格变化始终按同一个斜率变化。

以 Long Call 为例，当标的价格低于行权价时，期权不会被行权，买方最大亏损是权利金；当标的价格高于行权价后，期权开始产生内在价值，收益随着标的价格上涨而增加。

因此，期权收益图在行权价附近会出现折点，风险敞口会发生变化。这和期货不同。期货多头从一开始就是线性暴露，价格涨多少赚多少，跌多少亏多少。

所以，期权的非线性体现在：亏损和收益不对称，收益曲线有折点，Delta 会随着标的价格变化而变化。

## Part B：权利金到底是什么？

### Q4. 权利金为什么不是手续费？

我的回答：手续费是开平仓支付给交易所的费用，是交易的费用。而权利金支付的是买方未来有选择行权或者放弃行权的权利，权利金包含内在价值和时间价值

改进：

权利金不是手续费。手续费是交易过程中支付给交易所或经纪商的交易成本，而权利金是期权买方支付给期权卖方的价格。

买方支付权利金，是为了获得未来选择行权或放弃行权的权利。卖方收取权利金，是因为他承担了未来被行权的义务和不对称风险。

因此，权利金本质上是对期权这种非线性风险结构的定价。它不仅反映当前内在价值，也反映到期时间、波动率、市场预期、流动性和风险补偿。


### Q5. 期权买方为什么愿意支付权利金？

我的回答：支付固定的权利金，当未来标的价格朝不利自己的方向变动超出行权价时可以选择放弃行权，避免更大亏损，相当于最多亏损权利金这一有限的风险

改进：

期权买方愿意支付权利金，是因为权利金换来的是一种不对称的风险收益结构。

一方面，买方可以控制最大亏损。在不利情况下，买方可以放弃行权，最大亏损通常就是已经支付的权利金。

另一方面，买方可以保留有利方向的收益空间。比如买入 Call 后，如果标的价格大幅上涨，买方可以获得非线性收益；买入 Put 后，如果标的价格大幅下跌，买方可以获得保护或投机收益。

所以，期权买方支付的不是手续费，而是购买保护、选择权、时间和收益弹性。


### Q6. 期权卖方为什么愿意收权利金并承担义务？卖方到底在卖什么风险？

我的回答：期权买方虽然风险有限收益无限，但是买方盈利概率极低，卖方虽风险无限收益有限，但是概率上讲卖方胜率较高，卖方收取风险补偿但承担尾部风险

改进：

期权卖方愿意收取权利金，是因为他认为收到的权利金足以补偿自己承担的风险。

卖方表面上是在卖出一个 Call 或 Put，实质上是在卖出一种保险和风险承接能力。卖方承担的风险包括：标的价格大幅不利变动、波动率上升、Gamma 风险、尾部风险、流动性风险和保证金压力。

很多时候，卖方的胜率看起来较高，因为期权可能到期归零，时间价值会逐渐衰减。但这并不等于卖期权是低风险收益。卖方的收益通常有限，最大收益是权利金，而极端行情下可能一次亏掉长期积累的权利金收益。

因此，卖方赚的不是“免费钱”，而是承担风险后的风险补偿。

### 小结：期权价格不是只由方向决定，而是由方向、时间、波动率、行权价、到期时间和市场预期共同决定

时刻思考：
1. 我交易的是方向，还是波动率？
2. 我付出的权利金是否合理？
3. 我的风险是价格风险、时间损耗，还是波动率变化？

## Q1-Q6 阶段总结

通过 Q1-Q6，我对期权的基础理解是：

期货是线性风险工具，适合直接承担或对冲价格方向风险；期权是非线性风险工具，可以通过权利和义务的不对称设计风险结构。

期权买方支付权利金，获得未来在有利情况下行权、不利情况下放弃行权的权利，因此最大亏损通常有限。期权卖方收取权利金，承担未来被行权的义务，因此收益通常有限，但在极端行情下可能承担较大亏损。

权利金不是手续费，而是期权这种非线性风险结构的价格。它包含内在价值、时间价值、波动率预期、期限、流动性和市场风险补偿。

因此，期权不是简单的看涨看跌工具，而是风险定价、风险转移和风险结构设计工具。

## Part C：四种基础头寸

### Q7. Long Call 为什么不等于简单“看涨”？

我的回答：

long call属于强烈看涨，只有价格突破盈亏平衡点才会盈利，担当价格下跌时最多只会亏损权利金，风险是锁定的

改进：

| 维度    | Long Call      | 做多期货   |
| ----- | -------------- | ------ |
| 初始成本  | 支付权利金          | 占用保证金  |
| 最大亏损  | 通常为权利金         | 理论上较大  |
| 上涨收益  | 大涨时非线性受益       | 线性受益   |
| 下跌风险  | 亏损有限           | 跌多少亏多少 |
| 时间损耗  | 有              | 没有     |
| 波动率影响 | 有              | 没有     |
| 适合观点  | 看涨且预期上涨幅度/波动足够 | 直接方向判断 |


Long Call 不等于简单“看涨”。它确实表达上涨观点，但更准确地说，它是用有限权利金购买未来向上非线性收益弹性的结构。

直接做多期货是线性看涨，价格涨多少赚多少，跌多少亏多少；而 Long Call 的最大亏损通常被限制在权利金以内，但如果标的价格大幅上涨，收益可以继续扩大。

因此，Long Call 不只是判断方向，还要判断上涨幅度、上涨速度、到期时间、权利金是否合理以及隐含波动率变化。即使方向看对，如果标的上涨幅度不足以覆盖权利金，或者上涨太慢、时间价值损耗较大，买入 Call 也可能亏损。

从产业角度看，Long Call 可以用于未来采购风险管理。例如企业未来需要采购原材料，担心价格上涨，可以买入 Call 锁定采购成本上方风险。如果价格上涨，Call 获利以补偿采购成本上升；如果价格下跌，企业可以放弃行权，只损失权利金，并在现货市场以更低价格采购。

所以，Long Call 的核心不是简单看涨，而是用有限成本获得上涨保护、上涨弹性和非线性风险结构。

### Q8. Short Call 为什么风险很大？它最大的危险不是方向错，而是什么？

我的回答：

当标的价格突然大幅上涨后，所损失的是无限的，最大的危险时价格的突然剧烈拉升

改进：

改进版：

Short Call 风险很大，不只是因为方向看错，而是因为标的价格大幅上涨时，亏损会不断扩大，且可能伴随保证金压力、Gamma 风险、隐含波动率上升和流动性风险。

卖出 Call 的最大收益通常只是收到的权利金，但如果标的价格突破行权价并继续上涨，卖方需要承担被行权或平仓亏损的风险。价格越上涨，Call 的 Delta 越接近 1，卖方的风险暴露会越来越接近做空标的。

因此，Short Call 最大的危险不是普通小幅上涨，而是价格突然快速拉升、波动率上升、流动性变差和保证金压力同时出现。这也是裸卖 Call 在实盘中风险很高的原因。


### Q9. Long Put 为什么可以理解为“保险”？它和直接做空期货有什么区别？

我的回答：买入看跌，在持有标的资产多头头寸的时候，花费固定的权利金买入可以预防价格下跌的风险，在价格下跌的时候最多put option盈利，对冲标的下跌风险，而价格上涨最多只会亏损权利金

改进版：

Long Put 可以理解为保险，因为买方支付一笔确定的权利金，获得未来在价格下跌时的保护。如果标的价格下跌，Put 的价值上升，可以对冲现货库存或多头头寸的亏损；如果标的价格上涨，买方可以放弃行权，最大损失通常是权利金。

Long Put 和直接做空期货的区别在于：做空期货是线性对冲，会锁定价格风险，也会锁死价格上涨带来的收益；而 Long Put 是非线性保护，价格下跌时提供保护，价格上涨时仍然保留现货或多头头寸的上涨收益。

从产业角度看，钢贸商持有库存时，如果担心价格下跌但又不想完全放弃未来涨价收益，Long Put 比直接做空期货更灵活。但代价是需要支付权利金，并承担时间价值损耗和期权流动性风险。


### Q10. Short Put 为什么可以类比为“愿意低价接货”？这个类比的边界和风险在哪里？

我的回答：卖看跌，预期市场不会大跌或温和上涨，即认为价格在行权价附件已是最低，不会跌破行权价了，在市场大跌时亏损会很大

改进版：

Short Put 可以类比为“愿意低价接货”，是因为卖出 Put 后，卖方收取权利金，同时承担未来在标的价格跌破行权价时的买入义务。

如果标的价格没有跌破行权价，Put 可能到期归零，卖方赚取权利金；如果标的价格跌破行权价，卖方可能需要按照行权价承担买入标的或对应亏损。由于卖方提前收到了权利金，因此有效买入成本可以近似理解为：行权价 - 权利金。

但这个类比有严格边界。Short Put 只适合在卖方确实愿意在该价格买入，并且有足够资金和风控能力承受继续下跌风险的情况下使用。如果标的价格极端下跌，卖方会面临巨大亏损、保证金压力和流动性风险。

## Part D：真实交易理解

### Q11. 为什么买期权方向看对了，也可能亏钱？

我的回答：期权在期初时花了权力费，当价格上涨盈利无法覆盖权力费时则会导致亏损

我不确定的地方：都说期权买方随着时间流逝也会亏损，会有时间损益，那么这个损失的是期权的价格还是我买入期权的盈亏呢，是怎么计算的呢？

改进版：

买期权方向看对了也可能亏钱，因为期权盈亏不只取决于标的价格方向，还取决于上涨或下跌幅度、到期时间、时间价值损耗和隐含波动率变化。

买入期权后的持仓盈亏主要看期权价格变化，而期权价格由内在价值和时间价值组成。即使标的价格朝有利方向变化，如果幅度不够大，无法覆盖权利金，或者时间价值衰减、隐含波动率下降，期权价格仍然可能下降。

因此，买 Call 不是只要标的上涨就赚钱，而是要上涨得足够多、足够快，并且权利金买得不贵。买 Put 也是一样，方向正确但波动不够或时间损耗过大，仍然可能亏损。


### Q12. 为什么卖期权看起来胜率高，但长期可能因为一次极端行情亏很多？

我的回答：期权卖方必须履行义务，无论亏损多大都必须承接

改进版：

卖期权看起来胜率较高，是因为很多期权最终可能到期归零，时间价值会逐渐衰减，卖方可以通过收取权利金获得收益。

但卖方的问题在于收益有限、亏损可能很大。卖方最大收益通常是收到的权利金，而当标的价格出现极端不利波动时，亏损可能远远超过之前收取的权利金。

因此，卖期权不是稳定收租，而是承担尾部风险后的风险补偿。一次极端行情可能让卖方遭遇价格跳空、波动率上升、保证金追加和流动性风险，从而亏掉长期积累的权利金收益。


### Q13. 如果市场都知道某个事件要发生，为什么提前买期权不一定赚钱？

我的回答：期权价值等于内在价值加时间价值，期权买方会有时间损耗，只有波动率大才有可能突破盈亏平衡点赚钱，震荡也会亏损

改进版：

如果市场都知道某个事件要发生，提前买期权不一定赚钱，因为事件预期可能已经反映在权利金中。

事件发生前，市场预期未来波动会变大，投资者买入期权保护或投机，会推高隐含波动率，使期权价格变贵。此时买入期权，相当于用较高价格买入未来波动。

事件发生后，如果实际价格波动没有超过市场原先预期，或者事件落地后不确定性下降，隐含波动率可能快速回落，期权价格反而下跌。即使方向判断正确，如果波动幅度不足、时间价值下降或隐含波动率回落，买方仍可能亏钱。

因此，事件前买期权的核心不是判断事件是否发生，而是判断实际波动能否超过期权价格中已经隐含的波动预期。

## Q8-Q13 阶段总结


买期权不是只要方向判断正确就能赚钱，还需要标的价格在有限时间内产生足够大的有利变化，并且需要考虑权利金、时间价值损耗和隐含波动率变化。

卖期权也不是稳定收取权利金。卖方虽然可能在很多普通行情中获利，但本质上是在承担尾部风险、Gamma 风险、波动率上升风险、流动性风险和保证金压力。
