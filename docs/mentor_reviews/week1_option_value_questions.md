# Week 1 Option Value and PnL - Question List

## Part A：期权价格的组成

### Q1. 什么是期权的内在价值？

我的回答：内在价值是期权的行权价格与行权时标的资产市场价格之间的差

改进版：

期权的内在价值是指如果现在立即行权，期权能够带来的实际经济价值。

对于看涨期权 Call：

```text

内在价值 = max(标的价格 ➖ 行权价， 0)

```

对于看跌期权：

```text

内在价值 = max(行权价 ➖ 标的价格，0)

```


### Q2. 什么是期权的时间价值？

我的回答：权利金中超出内在价值的部分是时间价值

改进：

 时间价值本质上是市场为未来不确定性、未来变成实值的可能性、波动率和剩余期限支付的价格。


### Q3. 为什么说期权价格 = 内在价值 + 时间价值？这个公式在到期前和到期时有什么区别？

我的回答：因为期权价格是由内在价值和时间价值构成的，时间价值在到期时为0甚至为负，内在价值与时间价值均有可能为0

改进：

当前阶段，时间价值到期归零

## Part B：到期前盈亏 vs 到期盈亏

### Q4. 买入期权后，如果到期前平仓，盈亏应该看什么？

我的回答：

买入时期权的价格和现在期权的价格之差再减去期权费用

改进：

到期前平仓盈亏 = 当前期权价格 - 买入时的期权价格


### Q5. 如果持有期权到期，盈亏应该怎么算？

我的回答：对于买方来说是否由虚值期权转化为实值期权

改进：
如果持有期权到期，盈亏要看到期内在价值能否覆盖最初支付的权利金。

对于买入 Call：

```text
到期利润 = max(S_T - K, 0) - Premium
```


对于买入put：

```text
到期利润 = max(K - S_T, 0) - Premium
```

### Q6. 为什么不能简单用“标的涨跌幅”判断期权买方是否赚钱？

我的回答：因为期权买方购买的不仅是方向性的涨跌，还有波动率和时间价值，因为期权的价值等于内在价值+时间价值，会有时间的损耗，因此就算内在价值上涨时间价值下跌也有可能不赚钱

改进版：

不能简单用标的涨跌幅判断期权买方是否赚钱，因为期权价格不只由标的方向决定，还受到内在价值、时间价值、剩余期限、隐含波动率和市场预期影响。

买入 Call 后，即使标的上涨，如果上涨幅度不够大，无法覆盖权利金，或者时间价值衰减、隐含波动率下降，期权价格仍然可能下跌。

因此，买期权不是只判断方向，而是要同时判断方向、幅度、速度、时间和权利金是否合理。

## Part C：时间价值损耗

### Q7. 时间价值损耗损耗的到底是什么？是账户现金直接减少，还是期权市场价格下降？

我的回答：

期权价值=内在价值+时间价值，因此导致的是期权市场价格的下降


改进版：

时间价值损耗损耗的是期权价格中的时间价值部分。

它不是账户现金每天直接被扣掉一笔钱，而是因为到期时间逐渐减少，期权未来变得更有价值的机会减少，导致期权市场价格下降。对于买方来说，这会表现为持仓市值下降和浮动亏损；对于卖方来说，如果其他条件不变，时间价值衰减有利于卖方。

例如我用 5 元买入一个虚值 Call，如果标的价格不动、隐含波动率不变，随着时间流逝，这个 Call 可能跌到 4.7 元。我的现金没有直接少 0.3 元，但期权持仓市值下降了 0.3 元。


### Q8. 为什么越接近到期，虚值期权的时间价值通常衰减越快？

我的回答：

时间价值承载的是涨到目标价格的可能性，越接近到期虚值期权转变成实值期权的可能性越低，因此时间价值衰减的越快

改进版：

时间价值本质上是市场为未来不确定性和未来变成实值的可能性支付的价格。

虚值期权没有内在价值，它的价格几乎全部来自时间价值。越接近到期，标的价格在剩余时间内突破行权价并变成实值的可能性越低，因此虚值期权的时间价值会快速衰减。

例如标的价格为 100，行权价为 110 的 Call 是虚值期权。如果距离到期还有 3 个月，它仍然可能因为未来上涨而有较高时间价值；但如果只剩 1 天到期，标的仍在 100 附近，那么它变成实值的概率很低，时间价值会快速接近 0。

### Q9. 时间价值损耗对期权买方和卖方分别意味着什么？

我的回答：

对买方来说是亏损，且赚钱的概率越小；对卖方来说，赚取的一部分就是时间价值损耗的钱

改进版：

时间价值损耗对期权买方和卖方的影响相反。

对买方来说，时间价值损耗是不利因素。买方支付权利金买入未来变得有价值的机会，随着到期日临近，如果标的价格没有发生足够大的有利变化，期权的时间价值会下降，导致期权价格下跌，买方出现浮动亏损。

对卖方来说，时间价值损耗通常是有利因素。卖方收取权利金后，如果标的价格没有出现大幅不利波动，期权时间价值逐渐衰减，卖方可以通过期权价格下跌获利。

但卖方赚取时间价值并不等于低风险。卖方同时承担标的大幅波动、Gamma 风险、隐含波动率上升、流动性风险和保证金压力。因此，卖方赚的是承担风险后的时间价值补偿。

## Part D：方向对了也亏钱

### Q10. 买 Call 后标的上涨，为什么仍然可能亏钱？

我的回答：

标的上涨带动期权内在价值上涨，但时间价值下跌超过内在价值上涨的幅度也会亏钱

改进版：

买 Call 后标的上涨仍然可能亏钱，因为买方不仅需要判断方向，还需要判断上涨幅度、上涨速度、时间价值和隐含波动率变化。

如果标的只是小幅上涨，Call 的内在价值增加可能无法覆盖最初支付的权利金。到期时，只有当标的价格超过“行权价 + 权利金”这个盈亏平衡点时，Long Call 才真正盈利。

在到期前，即使标的上涨，如果时间价值衰减较快，或者隐含波动率下降，Call 的市场价格也可能低于买入价格，从而导致买方亏损。

因此，买 Call 不是只要上涨就赚钱，而是要涨得足够多、足够快，并且买入时权利金不能过贵。


### Q11. 买 Put 后标的下跌，为什么仍然可能亏钱？

我的回答：

也是时间价值的亏损

改进版：

买 Put 后标的下跌仍然可能亏钱，因为 Put 的盈利不只取决于方向，还取决于下跌幅度、下跌速度、时间价值损耗和隐含波动率变化。

对于 Long Put，到期利润为：

```text
到期利润 = max(K - S_T, 0) - Premium
```

### Q12. 如果事件发生前买入期权，为什么事件落地后可能亏钱？

我的回答：

可能在事件发生前实际波动率已经很高了，同时市场已经计价，隐含波动率处于高位，到事件发生时导致波动率坍塌，波动率越低期权价格下跌也会亏钱

改进版：

事件发生前买入期权不一定赚钱，因为事件预期可能已经反映在期权价格中。

事件前，市场知道未来可能出现大波动，很多投资者会买入期权进行保护或投机，这会推高隐含波动率，使期权权利金变贵。此时买入期权，相当于用较高价格购买未来波动。

事件落地后，如果实际价格波动没有超过市场原本预期，或者不确定性消失导致隐含波动率快速下降，期权价格可能下跌，这就是波动率坍塌。

因此，事件交易的关键不是“事件会不会发生”，而是“实际波动是否超过期权价格中已经隐含的波动预期”。如果没有超过，即使方向判断正确，期权买方也可能因为 IV 下降、时间价值衰减和权利金过高而亏损。

### 小结

## Q8-Q12 阶段总结

通过 Q8-Q12，我理解到，期权买方的盈亏不只取决于方向，还取决于幅度、速度、时间价值和隐含波动率变化。

时间价值本质上是未来可能性的价格。越接近到期，期权未来变得更有价值的机会越少，时间价值会逐渐衰减。对买方来说，时间价值损耗是不利因素；对卖方来说，时间价值损耗通常有利，但卖方同时承担 Gamma、Vega、尾部风险和保证金风险。

买 Call 时，标的上涨不一定盈利，因为上涨幅度可能不足以覆盖权利金，或者时间价值和隐含波动率下降导致期权价格下跌。买 Put 时也是一样，标的下跌不一定盈利，必须跌破盈亏平衡点才真正盈利。

事件前买期权也不一定赚钱，因为事件预期可能已经被市场计入权利金。事件落地后，如果实际波动没有超过隐含波动率所反映的市场预期，或者隐含波动率快速下降，期权买方可能亏损。
