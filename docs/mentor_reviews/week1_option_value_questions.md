# Week 1 Option Value and PnL - Question List

## Part A：期权价格的组成

### Q1. 什么是期权的内在价值？

我的回答：内在价值是期权的行权价格与行权时标的资产市场价格之间的差

改进版：

期权的内在价值是指如果现在立即行权，期权能够带来的实际经济价值。

对于看涨期权 Call：

$$

\text{内在价值}=\max(\text{标的价格}➖\text{行权价}，\ 0)

$$


### Q2. 什么是期权的时间价值？

我的回答：权利金中超出内在价值的部分是时间价值

改进：

 时间价值本质上是市场为未来不确定性、未来变成实值的可能性、波动率和剩余期限支付的价格。


### Q3. 为什么说期权价格 = 内在价值 + 时间价值？这个公式在到期前和到期时有什么区别？

我的回答：因为期权价格是由内在价值和时间价值构成的，时间价值在到期时为0甚至为负，内在价值与时间价值均有可能为0
